Selected Work

Four projects I keep coming back to. All on GitHub.

  1. Scanner Source Identification

    2026

    Identifies the source scanner of a TIFF image using sensor pattern noise. The pipeline extracts non-overlapping 1024×768 patches, denoises them, computes residual noise, builds a 16-dimensional feature vector, and trains an SVM. Reaches 88.44% accuracy across four scanner classes on 24,270 patches.

    Python · OpenCV · scikit-learn · PyWavelets · view on GitHub

  2. Secure UDP Chat — DRDO

    2025

    A single-file peer-to-peer chat application built during my DRDO internship. AES-256 in CBC mode, RSA-2048 key exchange, HMAC-SHA256 message integrity, and auto peer discovery over UDP — all wrapped in a Tkinter GUI.

    Python · sockets · Tkinter · cryptography · view on GitHub

  3. Sales Dashboard

    2026

    A Streamlit dashboard analyzing 10,000+ sales records — KPIs, regional performance, monthly trends, top products. The interesting part was automating the data-cleaning pipeline that used to be a manual chore: dedup, type coercion, category-name normalization, numeric conversion of currency fields.

    Python · Streamlit · pandas · Matplotlib · view on GitHub

  4. Diabetes Risk Prediction

    2026

    A small SVM model predicting diabetes risk from medical parameters (glucose, BMI, insulin, age, etc.), with feature standardization and a real-time prediction interface. Around 85% accuracy on the canonical Pima dataset — useful as a teaching project on the value of preprocessing and the limits of simple linear models.

    Python · scikit-learn · NumPy · pandas · view on GitHub
