# ayushkr80.github.io

Personal site for Ayush Kumar — Data and business analyst, IIT Bhilai '27.

Built by hand in plain HTML, CSS, and a little JS. No framework, no build step,
no dependencies you don't already have in a browser.

## Stack

- HTML, CSS, vanilla JS
- Inter (sans) and Fraunces (serif) from Google Fonts
- Hosted on GitHub Pages

## Files

- `index.html` — content
- `style.css` — styles
- `script.js` — mobile menu and active-nav scroll-spy
- `README.md` — this file

## Updating

Edit any of the four files directly through GitHub's web editor. Changes
go live in around 30 seconds via GitHub Pages.

The "Now" section at the top should be updated every couple of months —
that's the part that signals "this site is alive."

## Local development

Open `index.html` in any browser, or run a small local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
